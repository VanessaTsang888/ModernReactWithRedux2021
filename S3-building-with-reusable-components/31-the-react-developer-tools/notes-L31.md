# 31 The React Developer Tools

For Chrome and FireFox:

Chrome:
Search for React developer tools. Install this extension.

To use it:

1. Go to our Profile Card web app in the UI -> right click and select Inspect.
2. Click the right arrow to see the other tabs on the main menu, and click on Components tab.
3. Now we can see our different components that make up our application and they are organised in a tree like format.

Click on the different elments to see the props they been given - key-pair-value (data).

This tool is useful for debugging.

FireFox:
I use similar process to install the same tool but for FireFox browser.

I have installed this tool on both Chrome and Firefox browsers. See attached screenshots.
